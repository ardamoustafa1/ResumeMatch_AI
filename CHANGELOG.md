# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-06-21

### Added
- Secure cookie authentication, refresh rotation and logout revocation.
- Optional email verification and password reset functionality.
- Authorization for analyses, Telegram settings, uploads and WebSockets.
- Safe upload limits and authenticated PDF/image extraction.
- Standardized progress events and partial analysis results.
- Backend/frontend tests, migration checks and container CI.
- Security, privacy and contribution documentation.

### Changed
- Completed the Next.js migration and production container.
